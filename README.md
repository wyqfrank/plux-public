# Plux — AI-Powered Social Travel Planning Platform

## Overview

Plux is a IOS App that converts social media travel content (posts from Instagram, TikTok, etc.) into structured travel itineraries using large language models and geospatial APIs.

The platform was built to explore how AI can bridge the gap between travel discovery on social media and actual trip planning.

This repository documents the system design, architecture, and engineering decisions behind the project.
